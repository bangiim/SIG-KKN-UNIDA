# Instalasi
1. Buka command promp lalu ketikkan "
~~~
git clone https://gitlab.com/muhammadibrahim/sig-kkn-unida.git
~~~
2. masuk ke folder project
~~~
cd sig-kkn-unida
~~~
3. composer install
4. php artisan key:generate
5. php artisan migrate
6. php artisan serve
<br>
Project is ready.
